객체

* 객체 선언 및 호출

객체는 중괄호{}를 사용해서 생성하고,
중괄호 안에 이 객체에 필요한 속성들을 정의한다.
속성: 속성값의 형태로 정의함(키:벨류)
속성값으로는 모든 자료형의 값 들어갈 수 있음 

        var 변수명 = {
            속성: 속성값,
            속성: 속성값,
            ...
        }

* 객체 생성 특이케이스

* 객체에서의 반복문

객체가 가지고있는 모든 속성들에 접근하고자 한다면 반복문 이용 가능
단, 단순 for loop문으로는 사용 불가능하고 for in문만 사용 가능함

* 객체에서의 메소드 속성

객체의 속성 중 함수 자료형인 속성을 메소드라고 부른다 !!!

* in과 with 키워드

in : 객체 내부에 해당 속성이 있는지 확인해주는 키워드
with : 객체명을 생략할 수 있음 즉, 코드를 간결하게 줄여줄 수 있는 키워드
