window 객체

window 객체는 자바스크립트 상의 최상위 객체이며 크게 BOM과 DOM으로 나뉨
- BOM (Brower object Model) : location객체, screen객체, navigator객체, history객체
- DOM (Document object Model) : document객체

* Window 객체

- window.open("url", "창이름", "창의특성")


* window객체의 timer관련 메소드

- window.setTimeout(함수, ms)


- window.setInterval (함수, ms)
