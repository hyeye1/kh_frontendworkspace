목록관련 스타일 (list-style~)

list-style-type : 목록의 불릿기호를 변경시켜줄 때 사용

    * 순서 없는 목록 (ul) 
    선택자{
        list-style-type:disc(기본값)|circle|square|none;
    }

    * 순서 있는 목록 (ol)
    선택자{
        list-style-type:decimal(기본값)|decimal-leading-zero;
        list-style-type:lower-alpha|upper-alpha;
        list-style-type:lower-roman|upper-roman;
    }

list-style-position : 불릿기호의 위치를 조정할 때 사용
list-style-image : 불릿기호로 이미지를 적용하고자 할때 사용 

    선택자{
        list-style-position:outside(기본값)|inside;
        list-style-image:url(적용시키고자하는이미지경로);
    }
